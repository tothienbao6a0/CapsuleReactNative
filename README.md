## Getting Started

### Prerequisites

- Node.js >= 18
- npm or yarn
- iOS: XCode (for iOS development)
- Android: Android Studio (for Android development)

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd CapsuleApp
```

2. Install dependencies:
```bash
npm install
```

3. Install iOS dependencies:
```bash
cd ios
pod install
cd ..
```

4. Start the application:

For iOS:
```bash
npm run ios
```

For Android:
```bash
npm run android
```
